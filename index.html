# Créer une queue
curl -X POST https://miniqueue.vercel.app/api/queues \
  -H "Content-Type: application/json" \
  -d '{"name":"mon-projet"}'

# Ajouter un job
curl -X POST https://miniqueue.vercel.app/api/queues/mon-projet/jobs \
  -H "Content-Type: application/json" \
  -d '{"payload":{"task":"send_email","to":"user@example.com"}}'

FAQ

🤔 C'est quoi MiniQueue ?

Une API REST simple pour gérer des files d'attente de jobs. Vous créez une queue, vous ajoutez des jobs, et vous interrogez leur statut. Idéal pour les tâches asynchrones dans vos projets.

💾 Pourquoi SQLite et pas Redis ?

Redis est excellent mais nécessite une infrastructure séparée. SQLite permet à MiniQueue de fonctionner sans configuration — parfait pour les side projects et petites applications.

💰 C'est gratuit ?

Oui ! Un tiers gratuit est disponible. Pour plus de volume, les plans Pro ($5/mois) et Business ($15/mois) débloquent plus de jobs et de queues.

🔒 Comment sécuriser l'accès ?

Chaque requête nécessite une clé API. Vous pouvez générer des clés via le dashboard. Le rate limiting est intégré pour éviter les abus.
