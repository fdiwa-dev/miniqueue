📋 MiniQueue

L'API de file d'attente la plus simple pour les développeurs

Crée, priorise et traite des jobs en secondes. Pas de Redis, pas de configuration, pas de prise de tête.

🚀 Essayer la démo
⚡

Zéro complexité

Pas de Redis, pas de Docker, pas de Kafka. Une API REST, ça marche.

🎯

Priorité intelligente

Les jobs prioritaires passent devant. FIFO + priorité, le meilleur des deux mondes.

📦

SDK prêt à l'emploi

Client npm, docs claires, 5 lignes pour commencer. Compatible Node.js, browser et CLI.

📖 API en un coup d'œil

POST /api/queues

→ Créer une file d'attente

POST /api/queues/:id/jobs

→ Enqueue un job

POST /api/queues/:id/dequeue

→ Récupérer le prochain job

POST /api/jobs/:id/complete

→ Marquer complété

POST /api/jobs/:id/fail

→ Marquer échoué

GET /api/queues/:id/stats

→ Statistiques temps réel
